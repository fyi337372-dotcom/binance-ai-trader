# 🤖 AI 交易信号助手

基于币安API的技术分析工具，智能分析币种走势，提供交易建议。

## 🌐 在线访问

**网页版**: https://fyi337372-dotcom.github.io/binance-ai-trader/binance_web.html

## 📊 功能特点

- **实时价格查询** - 支持查询任意币种实时价格
- **技术指标分析** - MA20均线、RSI相对强弱指标
- **智能信号判断** - 根据多项指标综合给出买入/卖出建议
- **一键跳转购买** - 点击按钮直接跳转币安下单页面
- **快捷币种按钮** - 支持 BNB, BTC, ETH, SOL, XRP, DOGE, PEPE, SHIB, TRX, ADA, AVAX

## 🔧 技术栈

- 前端: HTML + CSS + JavaScript
- 数据源: Binance API (无需API Key)
- 部署: GitHub Pages

## 📈 指标说明

### MA20 (20小时均线)
- 计算过去20根1小时K线的平均值
- 价格 > MA20 表示多头趋势
- 价格 < MA20 表示空头趋势

### RSI (相对强弱指标)
- 数值范围 0-100
- >70 超买区域，可能下跌
- <30 超卖区域，可能上涨
- >50 多头区域
- <50 空头区域

## 🎯 信号评级

| 评分 | 建议 |
|------|------|
| ≥2 | 强烈买入 📈📈 |
| 1 | 轻度买入 📈 |
| 0 | 观望 ⚪ |
| -1 | 轻度卖出 🔴 |
| ≤-2 | 强烈卖出 📉📉 |
